Capstone Design

    Cellular Pump Controller

  • 3 Person group project; 2 Electrical Engineers; 1 Computer Engineer
  • Use of management and collaboration systems such as Jira, Bitbucket, Git
  • Design a pump controller using AWS (Amazon Web Services) that will report real time sensor data and pump status to AWS
  • Pump Controller must take analog voltage data from sensors and output results to AWS over a cellular connection using PPP(point to point protocol)
  • Responsibilities include: AWS interaction with Pump controller, cellular communication between controller and AWS, a web app
  • Beaglebone microcomputer running linux and Python for sensor interaction and AWS interaction
